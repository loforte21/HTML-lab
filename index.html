Introdução ao HTML

O que é HTML?

Linguagem de marcação para criar páginas web.

Elementos, tags e atributos

Elementos são blocos fundamentais, tags são marcas (<tag>), atributos fornecem informações adicionais.

Estrutura básica

<!DOCTYPE html>
<html>
  <head>
    <title>Titulo<title/>
  <head/>
  <body>
    <p>Paragrafo<p/>
  <body/>
<html/>

<!DOCTYPE html> define a versão HTML.

<html> é o elemento raiz

<head> contém informações do documento.

<title> define o título da página.

<body> contém o conteúdo visível.

Tags mais comuns

Textos em HTML

<p> para parágrafos.

<h1> a <h6>/code> para cabeçalhos.

Estilos de texto: <strong> (negrito), <em> (itálico).

Listas em HTML

<ul> para listas não ordenadas.

<ol> para listas ordenadas.

<li> para itens de lista.

Links em HTML

<a> cria hiperlinks.

Atributo 'href' especifica o destino do link.

Imagens em HTML

<img> insere imagens

Atributos 'src' (origem) e 'alt' (texto alternativo).

Formatação de Página

<br> para quebras de linha.

<hr> para linhas horizontais.

Comentários: <!-- ... -->.

Tabelas em HTML

<table> , <tr> , <td> para criar tabelas.

<th> define cabeçalhos de tabela.

Formulários em HTML

<form> inicia um formulário.

<input> , <textarea> para campos de entrada.

<button> para botões.

Multimídia em HTML

<audio> , <video> para incorporar áudio e vídeo.

<iframe> para conteúdo externo.
